Welcome to Course Tweaker

An Automated Course Recommendation System

Homepage & Dashboard

  • Users are directed to an active instrument panel upon login confirmation.
  • Dashboard contains charts and relevant items for making course selections utilizing Chart.js.

Course Recommendation Engine

  • Utilizes Udemy dataset for preparation and study.
  • Algorithms:
    • Cosine Similarity
    • Linear Similarity
  • Incorporates "neattext" library for Natural Language Processing (NLP) preprocessing.

Chatbot Integration

  • Machine learning-based chatbot trained using NLP methods.
  • Integration with Flask, PyCharm, and other libraries for logical flow.
  • Training and testing conducted in Jupyter notebooks for dataset refinement and model development.

Filtering and Feedback Loop

  • User preferences and feedback organized into the recommendation system.
  • Feedback loop for continuous improvement of recommendations.

Final Confirmation and Multiple Course References

  • Users presented with recommended courses.
  • Final confirmation allows users to select from various course options.
  • References link to detailed course information and enrollment.

Technological Stack:

  • Frontend: HTML, CSS, JS for UI
  • Backend: Flask for server-side development
  • Database: MySQL for storing user data
  • Machine Learning and NLP: Jupyter notebooks, scikit-learn, "neattext" library
  • Recommendation Engine: Custom algorithms using Cosine Similarity, Linear Similarity
  • Chatbot: NLP-based chatbot integrated with Flask
  • Dashboard: Chart.js for visualizations
